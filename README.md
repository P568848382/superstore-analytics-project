# Superstore Sales Analytics Project
## Overview
This project performs end-to-end business analytics on the Superstore dataset using Python, SQL, and Tableau.

The goal is to extract business insights related to sales performance, profitability, customer behavior, and operational efficiency.

## Key Business Insights

1. **Revenue Trend Analysis**
   - Analyzed sales growth over time.

2. **Top Performing Products**
   - Identified products contributing the highest revenue.

3. **Profitability Analysis**
   - Compared profit margin across categories.

4. **Geographical Sales Distribution**
   - Visualized sales across different markets.

5. **Discount Impact on Profit**
   - Identified how discounting affects profitability.

6. **Shipping Efficiency**
   - Analyzed delivery performance.

7. **Loss-Making Regions**
   - Identified regions with negative profit.
